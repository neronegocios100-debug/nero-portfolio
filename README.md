# Nero Portfolio

Projeto Next.js do portfólio com hero tipográfico e interação WebGL/prisma.

## Como abrir no notebook

1. Extraia este ZIP.
2. Abra a pasta `nero-portfolio` no VS Code.
3. Abra o terminal dentro da pasta.
4. Rode:

```bash
npm install
npm run dev
```

5. Abra no navegador:

```txt
http://localhost:3000
```

## Importante sobre as fontes

Eu não incluí os arquivos de fonte no ZIP.  
Coloque seus arquivos dentro da pasta:

```txt
public/fonts
```

Com estes nomes exatos:

```txt
Antipasto-Pro-ExtraLight-trial.ttf
Antipasto-Pro-Regular-trial.ttf
Antipasto-Pro-ExtraBold-trial.ttf
```

Se você não colocar as fontes, o site ainda abre, mas usa fonte fallback do sistema.
